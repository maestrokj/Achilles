Двухфакторная аутентификация v2

auth / security · фичи

Второй фактор при входе вдобавок к паролю — одноразовый TOTP-код из приложения-аутентификатора. Пользователь включает MFA из карточки «Безопасность» в профиле; вход в настройку защищён повторным вводом пароля. Мастер из трёх шагов подключает второй фактор, после чего двухфакторная аутентификация активна и управляется на том же экране.
