Почему features/ — доменные модули
Рядом с существующими shared-папками (components/, hooks/, lib/). Будущие модули → features/admin/, features/agents/ и т.д.
Запрещённые библиотеки — не использовать: python-jose (история CVE, слабое сопровождение), passlib (не сопровождается с 2020).
