v2 Отложенный охват

Тест-план выше покрывает scope первой версии. Зоны ниже проектируются вместе с соответствующими v2-фичами — здесь зафиксирован только охват, без детальных кейсов.

MFA TOTP (окно ±1 шаг, защита от повтора кода), recovery-коды, отдельный лимит MFA-шага, шифрование mfa_secret
SSO / OIDC state / PKCE / nonce, автопровижининг из IdP, авто-матч только по verified email
OAuth интерактивный поток выдачи токена, headless-вход через API-ключ
Мгновенный отзыв jti в Redis blacklist при деактивации — инвалидация без 15-минутного окна
