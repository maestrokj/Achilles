AI Foundation

сквозной слой · реестр AI-ресурсов платформы

Единый реестр AI-ресурсов платформы: провайдеры и их модели, каталог инструментов и системный prompt нейросетей — плюс управление ими: назначение моделей системным функциям, списки моделей чата и агентов, агрегат расходов. ORM-модели живут в общем ядре (core), Admin даёт к ним CRUD и экраны, а потребители — Harvester, Knowledge Store, Query Engine, Agent Engine — читают назначения отсюда, без циклов между модулями.
