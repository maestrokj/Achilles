активен истёк отозван
  1. 1 Открывает диалог создания — вспомогательная выдача ключа на сотрудника; основной путь — самовыпуск в профиле.
  2. 2 Фасеты Статус · Источник · Доступ. Поведение поиска и фильтров — единое для всех списков.
  3. 3 Владелец — ссылка на карточку пользователя с его ключами и доступами.
  4. 4 «Доступ» и «Источники» — скоуп ключа. Потолок прав — Auth & Security.
  5. 5 Owner / Admin отзывает любой ключ, отзыв применяется сразу. Механика — Auth & Security.
  6. 6 «отозван» — ключ погашен вручную или при деактивации владельца; реактивация его не возвращает.
