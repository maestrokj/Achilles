Пауза администратора
поставлена · снимается тем же тумблером
  1. 1 Личность агента — это личность владельца: агент читает Knowledge Store строго под его правами, запись в граф недоступна. Ссылка ведёт в карточку владельца. Agent Engine.
  2. 2 Конфигурация — имя, описание, промт, расписание, инструменты — только на чтение: правит её владелец в Web App.
  3. 3 Расход недели владельца против общего потолка — справочно. Сам потолок задаётся на экране Расходы AI.
  4. 4 Журнал прогонов — те же записи, что видит владелец: время, исход, длительность, токены, раскрываемый вывод. Agent Engine.
  5. 5 Единственный рычаг администратора над чужим агентом в v1 — липкая пауза с подтверждением; владелец её не снимет. Массовые действия и kill switch — v2. Agent Engine.
  6. 6 Инструменты — ядро KS (поиск, граф, SQL) плюс внешние, выбранные владельцем из разрешённых администратором; только на чтение, всё под личностью владельца. Допуск внешних инструментов задаётся в разделе Инструменты AI.
