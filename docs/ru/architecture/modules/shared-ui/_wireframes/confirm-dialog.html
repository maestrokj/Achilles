Confirm Dialog

shared ui · макеты · общий компонент

Подтверждение деструктивного действия. Три уровня по тяжести последствий.

Вариант Когда Тон
Reversible Состояние возвращается, данные целы Акцентная кнопка
Irreversible Данные или доступ уничтожаются Алерт + красная кнопка
Type-to-confirm Самое тяжёлое: удаление пользователя, организации Ввод имени разблокирует кнопку

Деактивировать пользователя? 1

Вход для Анны Петровой будет закрыт, активные сессии завершены. Учётная запись и история сохранятся — можно активировать обратно в любой момент. 2

  1. 1 Вопрос — глагол действия + объект, одна строка.
  2. 2 Последствия — что произойдёт; имя объекта выделено.
  3. 3 Отмена — ghost-кнопка, фокус по умолчанию.
  4. 4 Подтверждение — акцентная кнопка с guard-маркером, текст повторяет глагол (не «OK»).

Отозвать API-ключ?

Ключ ach_live_••••4f2a перестанет работать немедленно. Интеграции, использующие его, потеряют доступ.

Действие необратимо — отозванный ключ восстановить нельзя, понадобится создать новый.
