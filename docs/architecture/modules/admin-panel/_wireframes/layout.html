Section content (Outlet)
  1. 1 Theme (dark / light) and language (RU / EN) toggles. The org-default language lives in platform settings.
  2. 2 Notification center — a bell with an unread count (the current admin's unread in-app deliveries); clicking it opens the feed; channel and type configuration lives in notifications.
  3. 3 The avatar opens the profile menu, with sign-out of the current session and of all sessions.
  4. 4 The unread badge on the “Notifications” item rides the same push channel (SSE) as the bell. Badges on the other items are v2 — pending invites · sync errors.
  5. 5 Item visibility follows the role (Auth & Security): Owner sees everything; Admin sees everything except Audit Log, and Platform is read-only for them. The active item stays highlighted on a section's sub-screens too.
