  1. 1 Weekly token cap — one across all your agents; spend adds up across runs, and past the cap the agents wait for the weekly reset (Agent Engine). The cap itself is set by the administrator.
  2. 2 Clicking a card opens the agent's profile — configuration, manual run and run log. The switch and "Run" act in place, without opening the profile.
  3. 3 Summary status: "active" · "paused · admin" · "weekly limit" · "model removed" · "disabled". Combines the owner switch, the admin lock, the budget and the model's availability. Agent Engine.
  4. 4 Owner switch — an enabled agent runs on its schedule and via the button, a disabled one is stopped entirely. Under an admin lock ("paused · admin") the switch is unavailable.
  5. 5 Schedule — "manual", an interval or a calendar slot; edited in the profile. Agent Engine.
  6. 6 Last run — time, outcome, duration and token spend; the full output and history are in the profile.
  7. 7 "Run" — an ad-hoc run with the current settings; it respects the switch, the admin lock and the budget, so it's inactive for a disabled, limited or paused agent.
