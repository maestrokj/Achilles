Forced password change

auth / security · features

A user whose password an administrator has reset with no SMTP configured receives a temporary password. On the first login with it, the system won't let them into the app: it takes them to the change screen and holds them there until a permanent password is set. Only then does the role-specific workspace open.

Flow
Login shared · start · temporary password
signed in · must_change_password
Change Temp Password gate · new password
permanent password set
Role-specific app shared · finish · workspace
