### What it does

- **One AI hub** — connect providers (Anthropic, OpenAI, your own LLMs), turn models on, and assign them to jobs.
- **Spend in check** — track token usage per employee and per model, set limits, and get budget alerts.
- **Agents without code** — employees build their own agents in the UI; admins see every agent and stay in control.
- **Tools, centrally managed** — one place to enable and govern the tools that models and agents can reach: MCP servers, web search, and more, with per-scope control over who gets what.
- **Company knowledge** — connectors pull sources into a knowledge graph; answers cite their origin and respect access rights.
- **Surfaces out of the box** — web app, Slack bot, MCP, Telegram, and a public API.
